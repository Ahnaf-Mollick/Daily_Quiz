# Daily Quiz 📚

A Flutter quiz app with question and result screens.  
State management is handled using **Dart classes and objects** without using GetX, Bloc, Provider, or Riverpod.

## ✨ Features

- Multiple choice questions
- Answer checking
- Next question navigation
- Quiz submission
- Score calculation
- Multiple test results
- Restart quiz option
- Clean UI

---

## 🛠️ Built With

- Flutter
- Dart
- Material UI
- OOP based state management

---

## 🧠 State Management

Custom state management using:

- Classes
- Objects
- Data manager pattern

No external state management packages used.
